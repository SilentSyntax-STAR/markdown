# Markdown笔记（VSCode版）
本仓库是学习B站up主「青空の霞光」Markdown教程时，在VSCode中整理的学习笔记，包含基础语法、数学公式、排版技巧等核心内容，附实操示例和实用小妙招，适合Markdown新手快速上手。

## 📚 笔记结构
| 章节 | 核心内容 |
|------|----------|
| 基础操作 | 标题、加粗/斜体/高亮、代码块、列表、表格、超链接、图片插入等15+基础语法 |
| 数学公式 | 上下标、分数、开根号、特殊符号（希腊字母/集合/积分）、综合运算式等KaTeX语法 |
| 实用小妙招 | VSCode多位置批量加空格、图床使用避坑等实操技巧 |

## 🛠️ 推荐工具/插件
### VSCode必备插件
- Markdown All in One：一站式基础排版、预览、快捷键支持
- Markdown Preview Enhanced：增强预览（适配数学公式渲染）
- markdownlint：语法规范检查
- Paste Image：一键粘贴截图到笔记
- Markdown Code Block Runner：代码块运行测试

### 图床推荐
- 自用：PICUI图床（注意：偶有稳定性问题，建议备用其他图床）
- 备选：阿里云OSS/腾讯云COS（稳定）、PicGo+Gitee（免费）

## ✨ 核心亮点
1. 所有语法均附「输入+输出」示例，新手易对照实操；
2. 聚焦VSCode场景，包含专属快捷键、插件配置、批量操作小技巧；
3. 数学公式覆盖KaTeX常用语法（上下标、积分、希腊字母等），满足学习/科研排版需求；
4. 标注实操避坑点（如图床稳定性、转义字符使用）。
