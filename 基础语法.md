
# Markdown笔记（VSCode版）


## 一、基础操作

首先要在VSCode里面下载markdown的插件，这里提供几个建议：
Markdown All in One
Markdown Preview Enhanced
markdownlint
Paste Image
Markdown Code Block Runner

点击右上角这里可以实时预览

### 1.各级标题

（1）标题上下各空一行
（2）几级标题就用几个＃，最多可以六个
（3）＃跟文字之间要空一格

### 2.文本加粗

（1）法一：前后各两个星号：\*\*（你要加粗的内容）**
如果把 * 当作文本那就不要空格，一旦空格就会被加粗。
（2）法二：前后各两个下划线:\__（你要加粗的内容）__
记得切换到英文输入下划线
（3）法三：选中需要加粗的文字，使用VSCode快捷键ctrl+B

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|今天天气&emsp;\*\*非常好\*\*|今天天气 **非常好**|

### 3.文本斜体

（1）法一：前后各加一个星号：\*（你要变成斜体的内容）*，斜体不需要空格
（2）法二：选中需要变为斜体的文字，使用VSCode快捷键ctrl+I

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|今天天气\*非常好\*|今天天气*非常好*|
|\***今天天气非常好\*\*\*|***今天天气非常好***|

### 4.文本高亮

方法：前后各加两个等号

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\==今天天气非常好\==|==今天天气非常好==|

### 5.转义字符

若想让 \* 作为普通字符显示，而非触发斜体语法，可通过转义字符实现
方法：在 \* 前添加反斜杠\\

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
| \\\\ | \\ |
|今天天气\\\*非常好\\\*|今天天气\*非常好\*|

### 6.角标

#### （1）上角标

方法：两侧各加一个\^

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|x\^2^|x^2^|

#### （2）下角标

方法：两侧各加一个~

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|x\~2~|x~2~|

#### （3）注释

方法：文字[^角标内容]
&emsp;&emsp;&emsp;&emsp;&emsp;[^角标内容]：xxx
xxx会出现在文章的末尾，可以滑到底部查看

例：
markdown输入：
今天天气非常好\[^1]
\[^1]:来源于VSCode

markdown输出：
今天天气非常好[^1]
[^1]:来源于VSCode

### 7.代码块

#### （1）普通文字

①法一：英文输入法下按Esc下面1左面的键 `，把要变成代码块的内容包裹起来
②法二：选中需要变成代码块的内容，按下Tab键。注意前面要加入空行

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\`今天天气非常好\`|`今天天气非常好`|

#### （2）代码

方法：
\```加语言名称
代码内容
\```

例：C语言代码

markdown输入：
\```c
\#include<stdio.h>
int main(){
&emsp;&nbsp;printf("hello world\n");
&emsp;&nbsp;return 0;
}
\```
markdown输出：

```c
#include<stdio.h>
int main(){
    printf("hello world\n");
    return 0;
}
```

### 8.空格

（1）`&nbsp;` 半角
（2）`&emsp;`全角
emsp ≈ 2个nbsp，具体根据排版调整

### 9.中划线

方法：前后各加两个波浪线：\~\~（你要划掉的内容）~~

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\~\~今天天气非常好~~|~~今天天气非常好~~|
|\*\*\*\~\~今天天气非常好~~***|***~~今天天气非常好~~***|

### 10.分割线

方法：在某空行输入三个或三个以上连续的星号***或下划线___,后面不能再输入文字

例：
markdown输入：\*\*\*
markdown输出：
***

### 11.列表

#### （1）简单列表

方法：在某空行输入星号*加空格，或者加号+加空格，或者减号-加空格-(同一列表内不可混合使用)。同时上下各空一行。

例：
markdown输入：

\* 任务一

\* 任务二

\* 任务三

markdown输出：

* 任务一

* 任务二

* 任务三

#### （2）数字列表

| 列表类型 | 操作方法 |
| :---: | :---: |
|一级列表|序号加小数点加空格|
|二级列表|四个空格加星号\*加空格|
|三级列表|八个空格加星号\*加空格|
|多级列表|以此类推|

例：
markdown输入：(无序列表+有序列表混合)

1\. 任务一：学习markdown
&emsp;&nbsp;&nbsp;&nbsp;\* 学习方法
&emsp;&nbsp;&nbsp;&nbsp;\* 学习计划

&emsp;&emsp;&emsp;&emsp;&emsp;&nbsp;\* 目标1
&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;\* 子目标1
&emsp;&emsp;&emsp;&emsp;&emsp;&nbsp;\* 目标2
&emsp;&emsp;&emsp;&nbsp;&nbsp;\* 第二天
2\. 任务二：学习C语言
&emsp;&nbsp;&nbsp;&nbsp;1\. 学习方法
&emsp;&nbsp;&nbsp;&nbsp;1\. 学习内容
&emsp;&emsp;&emsp;&nbsp;&nbsp;1\. 数组
&emsp;&emsp;&emsp;&emsp;&emsp;&nbsp;1\. 一维数组
&emsp;&emsp;&emsp;&emsp;&emsp;&nbsp;2\. 二维数组
&emsp;&emsp;&emsp;&nbsp;&nbsp;2\. 指针

markdown输出：

    * 学习方法

        * 第一天
            * 目标1
                * 子目标1

#### （3）列表打勾

方法：方括号中间①加空格是未勾选②空格变成x或X是对勾。方括号后面要加空格

例：
markdown输入：
\* [x] 学习markdown
\* [&nbsp;&nbsp;] 学习C语言

markdown输出：

* [x] 学习markdown
* [ ] 学习C语言

### 12.引用

作用：表示这里是引用别人的内容
方法：尖括号>加空格
可以与列表，代码块等互相嵌套

例：
（1）引用中嵌套其他
markdown输入：
\> 任务一
\>
\> \* 学习markdown
\>
\> 任务二
\>
\> \* 学习C语言

markdown输出：
> 任务一
>
> * 学习markdown
>
> 任务二
>
> * 学习C语言

(2)其他中嵌套引用
markdown输入：

\* 任务一
&emsp;&nbsp;&nbsp;&nbsp;\> 学习markdown
\* 任务二
&emsp;&nbsp;&nbsp;&nbsp;\>学习C语言

markdown输出：

* 任务一
    > 学习markdown
* 任务二
    >学习C语言

### 13.超链接

法一：\[文字](网址)
法二：\[文字][变量名]

&emsp;&emsp;&emsp;\[变量名]:网址

例：
markdown输入：
我把\[豆包]\[a]作为AI助手，\[CSDN]\[b]作为学习平台。

\[a]:`https://www.doubao.com`
\[b]:`https://www.csdn.net/`

markdown输出：
我把[豆包][a]作为AI助手，[CSDN][b]作为学习平台。

[a]:https://www.doubao.com
[b]:https://www.csdn.net/

### 15.插入表格

（1）基本形式：英文状态下的竖线| 文本 |
（2）分割线：减号| ---- |，个数随意
（3）左对齐：| :----|
（4）右对齐：| ----: |
（5）居中：| :----: |
表格前要空一行

例：
markdown输入：
\| 姓名 | 年龄 | 性别 |
\|:-----\: |:-----\: |:------\: |
\| 小明 | 18 | 男 |
\| 小红红 | 17 | 女 |
\| 打不死的小强 |16 | 男 |

markdown输出：

| 姓名 | 年龄 | 性别 |
|:----:|:----:|:----:|
| 小明 | 18 | 男 |
| 打不死的小强 |16 | 男 |
| 小红红 | 17 | 女 |

## 二、插入数学公式

### 1.基础操作

（1）将字母变为数学公式：
①单行：前后各加一个$
②多行：起始行和末尾行各加两个\$
（2）换行：在末尾输入两个反斜杠\\\

### 2.上标

方法：遵循KaTeX语法，只需要一个^。嵌套需要用大括号。

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\$x^{2a}\$|$x^{2a}$|

### 3.下标

方法：下划线\_

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\$x_1\^2+x_2^2=1\$|$x_1^2+x_2^2=1$|

### 3.分数

方法：\frac{分子}{分母}

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\$\frac{x+1}{2}\$|$\frac{x+1}{2}$|

### 4.开根号

方法：①平方：\sqrt{数字或计算式}
&emsp;&emsp;&emsp;②n次方：\sqrt[n]{数字或计算式}
如果在计算式中想要打出花括号\{}，需要加上转义字符\\。

例：

| markdown输入 | markdown输出 |
| :---: | :---: |
|\$\sqrt[3]{(4x-2)-3}\$|$\sqrt[3]{(4x-2)-3}$|
